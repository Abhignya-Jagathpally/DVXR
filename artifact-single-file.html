THE PRODUCT THESIS

Stress becomes clinically useful only when it changes a decision.

Sentinel does not stop at detecting stress. It tests whether neural and autonomic state add measurable, calibrated value to near-term glucose-risk forecasting, then translates the result into a constrained next action.

LIMITATIONS

THE EVIDENCE CONTRACT

Every visible conclusion has a designated source of authority.

The interface is intentionally organized so that exact facts, numerical inference, semantic retrieval, and generated language cannot silently substitute for one another.

01

SOURCE OF TRUTH

Structured records

Exact glucose values, timestamps, medications, permissions, consent, model outputs, and audit states remain in transactional or time-series stores.

Never delegated to vector similarity
02

NUMERICAL AUTHORITY

Predictive models

Modality-specific encoders and calibrated fusion calculate risk, forecast, confidence, uncertainty, or a formal abstention.

Never recalculated by the language layer
03

SEMANTIC CONTEXT

Vector retrieval

Approved protocols, de-identified note sections, model cards, and explanatory evidence are retrieved with patient, date, role, and version filters.

An index, not the source of truth
04

COMMUNICATION

Grounded language

A constrained LLM or SLM translates immutable structured outputs and retrieved evidence into an understandable review.

Validated before display
